SupermarioSpel - Kul!

INFORMATION

Controls - Player 1

Run forward - Right Arrow

Run backwards - Left Arrow

jump - Up Arrow

Controls - Player 2

Run forward - D

Run backwards - A

jump - W

Win the game!

Win the game by finishing the map before the time is over!

But be aware, the game will not be a walk in the park, as there is

many monsters patrolling the map!

Navigate through the different menus with up and down arrow, and select an option with enter.
